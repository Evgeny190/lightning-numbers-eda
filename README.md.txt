# Lightning numbers EDA

Exploratory Data Analysis (EDA) project focused on structuring, cleaning, and organizing raw data with Python.

## Project overview
This project demonstrates how to:
- inspect a dataset
- clean and structure columns
- work with dates
- sort and group data
- prepare data for further analysis

## Tools used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Files
- `EDA_structuring_with_Python.ipynb` — main notebook with the full analysis
- `dataset.csv` — source dataset used in the notebook

## Key steps
1. Load the dataset
2. Inspect dataframe structure
3. Clean missing or inconsistent values
4. Convert data types
5. Work with datetime columns
6. Sort, filter, and group data
7. Summarize findings

## Outcome
The notebook shows a practical example of how raw data can be transformed into a more structured and analysis-ready dataset using Python.
